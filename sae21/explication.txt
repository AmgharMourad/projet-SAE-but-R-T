Je suis devenu compétent pour " Administrer les réseaux et l'internet" car je suis devenu capable d'acquérir l'AC3011 qui consiste à configurer les fonctions de base du réseau local. En effet j'ai pu configurer pour une machine les IP de ses différentes interfaces sous linux. De plus, j'ai pu aussi configurer les interfaces IP de divers interfaces mais cette fois-ci sur un routeur. Pour cela j'ai pu m'appuyer par exemple sur le ressource R201 Technologies de l'Internet. Cela m'a permis d'apprendre comment bien configurer des interfaces pour que la communications entre différentes machines puisse passer correctement sans problème. En effet dans le cour nous avons appris à configurer les IP des différentes interfaces par exemple pour un routeur en appliquant les commandes "ip add addresse_IP masque_de_sous_réseau" ou même en créant des routes pour accéder à d'autre réseau en appliquant la commande " ip route adresse_réseau_destination masque_de_sous_réseau adresse_du_prochain_routeur" et nous avons donc appliquer ces commandes durant la SAE 21. Pour une machine sous linux, il suffit d'accéder à fichier qui se nomme "interfaces" et de modifier les IP. Je suis donc maintenant capable de configurer un réseau, mais il me reste encore à acquérir pour pourquoi pas configurer un réseau pour une entreprise.
